## ✨ What is TeamLegacy OS?

**TeamLegacy OS** is a fully interactive **browser-based operating system** that runs entirely in your web browser. Think Windows 95 meets cyberpunk future — but with modern 3D graphics, smooth animations, and real functionality.

Boot it up, explore the desktop, launch apps, play games, take notes, and dive into a retro-futuristic computing experience — no installation required.

**Try it free right now** → [Launch TeamLegacy OS](https://2ndchances520.github.io/TeamLegacy-OS/)

## 🔥 Features

### 🎮 Entertainment
- **2 Full 3D Games** — immersive experiences right in the OS
- Dynamic desktop backgrounds & themes
- Retro sound effects & startup chime

### 🛠️ Productivity
- **Notes App** — persistent note-taking with save/load
- **Terminal** — powerful command-line interface (try `help`, `neofetch`, `games`...)
- File Explorer with drag & drop
- Calculator, Clock, Settings

### 🎨 Customization
- Change themes, wallpapers, and window styles
- Resizable & draggable windows
- Taskbar with live clock

### ⚡ Technical
- Built with modern web technologies
- Responsive design (works great on desktop & tablets)
- Offline capable (PWA ready)
- Smooth 60fps animations

## 🚀 Quick Start

1. **Just click and go** — Visit the [live demo](https://2ndchances520.github.io/TeamLegacy-OS/)
2. Click the big **Power Button** on the boot screen
3. Explore the desktop!
4. Right-click for context menu
5. Double-click icons to launch apps

**Pro tip:** Open the Terminal and type `help` for secret commands! 👾
